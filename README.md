\# 🧪 Chemical Pathology Diabetes Analysis



A beginner-friendly exploratory data analysis (EDA) project using the \*\*Pima Indians Diabetes Dataset\*\*.



This project marks the beginning of my data science journey, where I explore statistical concepts using real healthcare data before moving into predictive modeling and machine learning.



Rather than jumping straight into building models, I wanted to understand the data first—asking questions, interpreting results, and strengthening my statistical foundation.



\---



\## 📖 Project Overview



The project explores the Pima Indians Diabetes Dataset using Python and applies fundamental statistical concepts to understand the characteristics of the data.



The goal is not to build the most accurate predictive model, but to develop an intuition for how data behaves before introducing machine learning.

\## 📈 Dataset



The project uses the \*\*Pima Indians Diabetes Dataset\*\*, collected by the National Institute of Diabetes and Digestive and Kidney Diseases (NIDDK).



The dataset contains medical information from female patients of Pima Indian heritage and is commonly used for learning statistics, data analysis, and machine learning.
